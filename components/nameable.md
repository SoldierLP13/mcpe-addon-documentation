
# nameable

Can entity be named

## Parameters

* `alwaysShow: boolean` - Nameplate always visible
* `allowNameTagRenaming: boolean` - Disallow renaming 

## Example
From player and npc json files

````json
{
    "minecraft:nameable":{
        "alwaysShow": true,
        "allowNameTagRenaming": false
    }
}
````

other files

````json
{
    "minecraft:nameable":{}
}
````
