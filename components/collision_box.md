# collision_box

This component sets the volume that is collidable for the entity.

## Parameters

 * `width: float` - Width of the box (both x and z axis)
 * `height: float` - Height of the box

## Example

````json
{
    "minecraft:collision_box":{
        "width": 0.6,
        "height": 1.8
    }
}
````
